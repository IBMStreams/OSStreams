Samples for the Java Operator API.

Java source files and compiled classes for samples are contained in
$STREAMS_INSTALL/lib/com.ibm.streams.operator.samples.jar

A SPL sample toolkit for Java operators is provided in
{@code $STREAMS_INSTALL/samples/spl/feature/JavaOperators}
Within the toolkit sample Java primitive operators are provided using the Java Operator API samples within these SPL namespaces:

  • {@code com.ibm.streams.javaprimitivesamples.operators} - Java primitive operators from the {@link com.ibm.streams.operator.samples.operators} package
  • {@code com.ibm.streams.javaprimitivesamples.sources} - Java primitive operators from the {@link com.ibm.streams.operator.samples.sources} package
  • {@code com.ibm.streams.javaprimitivesamples.windows} - Java primitive operators from the {@link com.ibm.streams.operator.samples.windows} package
  • {@code com.ibm.streams.javaprimitivesamples.sinks} - Java primitive operators from the {@link com.ibm.streams.operator.samples.sinks} package
